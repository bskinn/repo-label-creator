## CHANGELOG: repo-label-creator -- Apply labels to GitHub repos

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and this project follows an extension of
[Semantic Versioning](https://semver.org/spec/v2.0.0.html), where a bump in a

### Not-Yet-Versioned

The initial work on this project involved a set of labels
in `labels.json` and a CLI script `create_labels.py` to
apply them to a repository. It also included an initial
schema for `labels.json`, in `schema.json`.
